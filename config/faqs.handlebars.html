Intelligent Flight:

There are different intelligent flight modes that make flying easier on the operator. For example: “Follow Me” mode will follow the operator from a fixed distance.


Gimbal:

A gimbal is a pivoted support that allows the rotation of an object about a single axis.


Headless Mode:

This is another feature to make flying easier. “Headless” implies you don’t need a brain to operate the drone.


Avoidance:

Obstacle avoidance, it does just that, avoids obstacles.


FPS:

Frames per second. The higher the FPS, the smoother a moving image appears.
